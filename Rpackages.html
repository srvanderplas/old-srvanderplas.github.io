Interesting R Packages

This page contains a list of R packages that are interesting or have the potential to be useful to me at a later date. Main page.

R packages

  • vembedr: Embed videos from YouTube and Vimeo in Rmarkdown documents. Use ALL the puppy videos!

  • rvest: A BeautifulSoup-like package for R.

  • USAboundaries: A package with US state/territory boundaries, counties, congressional districts, and zip codes from 1629 to 2000.

Shiny packages:

  • shinyTable: implements Handsontable for shiny, providing a spreadsheet-like editor in a web browser (data can then be read into R)

  • shinyRGL: provides webGL bindings, so that 3D graphics can be manipulated within shiny apps. Somewhat more difficult to set up the shiny-server backend, but still a very neat effect.

  • shinyBS package: Make bootstrap work nicely with Shiny and allows modal windows that are overlays. Good for documentation - help files show up elsewhere.

  • leaflet-shiny: adds shiny bindings for Leaflet.js, which makes mobile-friendly interactive maps

  • shiny-gridster: use the gridster library with shiny. Drag-and-drop multi-column grid layouts for web pages.

  • shiny-jsdemo: R shiny bindings for gridster, justgage, and highcharts. Calls the shiny-gridster package as a dependency. Also includes a status panel.

  • shiny-sky: Includes styled buttons and alerts not present in default shiny apps. Also adds event buttons and busy indicators, and includes handsontable and jsTree bindings. Adds color input as well.
